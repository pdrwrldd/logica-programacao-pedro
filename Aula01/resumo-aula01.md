O que é Git e por que vamos usar na disciplina? Git é uma ferramenta para controlar a versões de seus projetos. Com o Git, é possível registrar todas as mudanças feitas no projeto. Com isso, pode ajudar a ver o que foi alterado no projeto e tambem poder voltar para versões anteriores caso seja preciso. O Git é bastante utilizado na programação, pois facilita o trabalho em equipe e a organização do codigo. Em nossa aula, iremos usar a ferramenta para para guardar nossas atividades, acompanhar mudanças feitas na hora e manter o projeto mais organizado.
