Definição de algoritmo: Seria uma sequencia logica de ações para resolver um problema.

entrada: informações que são processadas de inicio nos algoritmos.

processamento: dados necessarios para o andamento de funcionamento do algoritmo.

Saida: finalização/resultado do andamento do processamento para resolver o problema.

exemplo algoritmo: escovar os dentes
- pegar a escova
- aplicar pasta de dente na escova
- fazer movimentos circulares
- cuspir a pasta
- enxaguar a boca
- enxaguar a escova

palavras-chave da aula
Algoritmo: instruções para executar ações afim de resolver um problema.

Hardware: Componentes fisicos utilizados para o funcionamento do computador.

IA: Maquinas capazes de simular processos humanos e executar tarefas>

Software: Sequencia logica para processar a funcionalidade e solucionar problemas de um computador.

Logica de programação: Raciocinio de sequencia finita criado para executar uma determinada tarefa.
